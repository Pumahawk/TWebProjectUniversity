#Framework php e sito internet ecommerce

##Framework

###Introduzione

Il framework php si ispira a Laravel, infatti si possono trovare molte componenti
simili come la classe Router, Middleware e la struttura delle cartelle.

###Cartelle

* *app* Cartella contenete il codice sorgente dell'applicazione.
* *bin* Cartella contenete il codice sorgente del framework a cui si appoggia l'applicazione.
* *public* Cartella pubblica accessibile dal client.
* *resource* Cartella contenente i file generati dall'applicazione.

###Vantaggi

Utilizzare un framework php permette di avere un'applicazione ordinata che può contare
sull'api messe a diposizione dal framework e sul suo pattern mvc. Grazie alla sua struttura
obbliga lo sviluppatore dell'applicazione a organizzare il codice sorgente in modo
che uno sviluppatore diverso possa avere un idea del suo funzionamento senza troppe
complicazioni.

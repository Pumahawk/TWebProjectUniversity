# Bin

La cartella bin contine tutte le classi che permettono corretto funzionamento del
framework più una collezione di api che facilitano la gestione della applicazione.

Si consiglia di mantenere invariato il contenuto di questa cartella.

###File

* *BaseMiddleware.php* File contenente i metodi per il corretto funzionamento 
	della classe Middleware.
* *BaseRouter.php* File che permette il corretto funzionamento della classe Router.
* *Database.php* File contente la classe Database che permette una facile connessione
	al database.
* *Framework.php* File contenente la classe Framework, che permette il funzionamento base 
	del framework.
* *Resource.php* File contenente le api per la gestione dei file creati dall'applicazione.
* *Session.php* File contenente le api per la gestione della sessione utente.
